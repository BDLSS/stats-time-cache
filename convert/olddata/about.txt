This folder contains the data from Piwik as exported from
the existing database.

Method of export
----------------
Copy export.sh to piwik server.
Chmod u+x if needed.
Run, should output three files in TSV format in UTF-8

Expect results
--------------
These the result on 22nd January 2014:

piwik_log_link_visit_action has 1.52 million rows giving a 171MB file.
piwik_log_visit has 0.4 million rows giving a 153MB file.
piwik_log_action has 0.3 million rows giving a 33MB file.
